# EZFit (In Development)

EZFit is a web application currently under development that aims to simplify fitness tracking, planning, and personalization using modern web technologies and AI.

---
![App Screenshot](https://i.imgur.com/dhWyqHG.png)

## Tech Stack

### Frontend

* React
* HeroUI (for modern and responsive UI components)

### Backend

* Django REST Framework (for API and authentication)
* Custom Django API (ezfitAPI) for user login and authentication handling

### Database

* Azure SQL Database

### AI Integration

* Gemini API (for intelligent features and recommendations)

---

## Features (Planned)

* User authentication and secure login system via custom ezfitAPI
* Personalized diet plan generation based on user input (weight, height, goals, etc.)
* Macronutrient and calorie target customization
* Workout planner page for creating and managing training routines (In future)
* AI-assisted recommendations using Gemini API (In progress)
* Meal planning and tracking
* Clean and responsive UI

---


## Goals

* Build a reliable and scalable fitness platform
* Integrate AI to enhance user experience
* Provide a clean and intuitive interface
* Work with Azure SQL DB
* Combine both frontend and backend
